# Game Development Technology Analysis

A comparative analysis of hardware, software and cost requirements for producing a mobile indie game.  
The project evaluates two production setups: an economical configuration and a professional high-end configuration.

## Scope of the project

- specification of hardware configurations  
- comparison of software tools (free vs commercial)  
- cost estimation for workstations and licenses  
- workflow diagrams of the production process  
- analysis of security options and data backup strategies  
- assessment of scalability and possible extensions
