# Computer Network Configuration (Cisco Packet Tracer)

A complete LAN/WLAN network configuration created in Cisco Packet Tracer.  
The project includes routing, wireless setup, server configuration, firewall rules and diagnostic testing.

## Scope of the project

1. Network topology  
   - router, switch, PCs, laptops  
   - wireless printer  
   - central server providing multiple services

2. Device configuration  
   - router IP setup  
   - WPA2 wireless network  
   - static IP configuration for all clients  
   - printer Wi-Fi module configuration

3. Server services  
   - HTTP web page  
   - DNS domain resolution  
   - FTP access  
   - email server with two mail accounts

4. Security and filtering  
   - firewall rule blocking access for a selected client  
   - test confirming denied access

5. Diagnostics  
   - ping tests between all nodes  
   - traceroute  
   - ARP table inspection

## Purpose

The project demonstrates basic practical networking skills:
addressing, routing, wireless configuration, service deployment and verification using diagnostic tools.
