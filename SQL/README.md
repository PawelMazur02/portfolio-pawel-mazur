# SQL Projects

This folder contains my SQL and database-related work:

1. **Relational Car Dealership Database (OLTP)**  
   - SQL script with full schema (tables, PK/FK) and realistic sample data  
   - Entities: Cars, Customers, Employees, Orders, Service  
   - Used as a base for reporting and for the data warehouse project

2. **Data Warehouse & OLAP Cube for Car Dealership**  
   - Star schema with fact table `FactZamowienie` and dimension tables  
   - Implemented in MS SQL Server + Analysis Services (SSAS)  
   - OLAP cube created and deployed, basic browsing and analysis

3. **Distributed Databases – Cinema & Warehouse**  
   - Two SQL Servers (central and branch)  
   - Triggers, stored procedures, distributed transactions  
   - Synchronisation of prices and stock between servers

4. **SQL Assignments (EMP / DEPT)**  
   - Joins, GROUP BY, HAVING, subqueries, self-joins  
   - Salary and department reports, HR-style analysis

5. **Algorithms – GCD/LCM**  
   - Python implementation of Euclidean algorithm  
   - Basic and validated version

Each subfolder includes scripts and documentation related to the topic.
