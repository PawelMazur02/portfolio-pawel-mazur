# Car Dealership SQL Database (OLTP)

A simple relational database created for a car dealership system.  
Project shows basic SQL design: tables, relationships, sample data and business logic.

## Main Features
- Two schemas: **salon** (cars, orders, service) and **pospolstwo** (customers, employees)
- Proper relational structure (PK/FK)
- Realistic example data
- Simple modelling of a dealership’s workflow

## ncluded Tables
- **Cars** (Samochody)  
- **Customers** (Klienci)  
- **Employees** (Pracownicy)  
- **Orders** (Zamowienia)  
- **Service** (Serwis)

## What this project demonstrates
- Creating SQL tables  
- Designing relationships  
- Inserting and testing sample business data  
- Understanding basic OLTP structure
