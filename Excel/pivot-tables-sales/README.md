# Sales Analysis with Pivot Tables (Excel)

This project contains several analytical reports built using Excel Pivot Tables.  
The goal was to explore sales data and prepare clear summaries based on products, sellers and locations.

## Scope of the project

1. Pivot table summarizing car sales  
   - Breakdown by seller, product model and city  
   - Total sales value and quantity  
   - Percentage share per seller

2. Pivot table for office supplies dataset  
   - Interactive filtering using a slicer (city)  
   - Aggregation of quantities per product

3. Sales summary with discounts  
   - Total sales, discount value and post-discount sales  
   - Grouped by product category

## Purpose

The project demonstrates the ability to:
- organise raw data for reporting,
- build Pivot Tables for different business questions,
- use slicers and formatting,
- summarise sales performance in a clear analytical form.
