# Excel Formulas – Analytical Examples

This project contains several practical exercises demonstrating core analytical functions in Excel.  
The workbook includes examples of conditional calculations, lookups, logical formulas and array-based computations.

## Scope of the project

1. Conditional counting and summing  
   - COUNTIF  
   - COUNTIFS  
   - SUMIF  
   - SUMIFS  

2. Lookup operations  
   - VLOOKUP used for retrieving prices, trainer rates and other reference data  

3. Logical formulas  
   - Nested IF for classifying values into categories  
   - Simple decision rules for scoring and evaluation  

4. Array-based calculations  
   - SUMPRODUCT for computing values such as total sales based on quantity and unit price  

5. Conditional formatting  
   - Highlighting values based on thresholds  
   - Category-based color rules  

## Purpose

The project demonstrates the ability to work with essential Excel functions used in reporting and business analytics, including filtering, classification, lookups and numerical summarisation.
